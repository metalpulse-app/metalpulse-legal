Security

Built secure.
From the ground up.

VPS IPs are sensitive. MetalPulse treats them that way.

✓ iOS Keychain · Android Keystore — all node configs encrypted at rest
✓ SSRF protection — every RPC URL validated before it hits the network
✓ Timeouts everywhere — no hanging requests, no resource exhaustion
✓ zod schema validation on every external API response
✓ HMAC integrity check — tamper-proof store on rooted devices
✓ IPv6 ULA & link-local blocked — full private-range SSRF protection
